# nettrace

Dashboard Streamlit para mapear conexões IP/MAC no tempo

## Execucao

```bash
streamlit run main.py
```

## Estrutura

- `main.py`: ponto central de execucao e navegacao.
- `pages/IP.py`: visao com selecao de IP e eixo Y por MAC.
- `pages/MAC.py`: visao com selecao de MAC e eixo Y por IP.
